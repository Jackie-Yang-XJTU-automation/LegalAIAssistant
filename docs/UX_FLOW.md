# UX Flow — 律师私人案件助理

## 新用户首次使用流程

1. 打开应用 → 看到空白首页（空状态引导）
2. 点击"创建第一个案件"
3. 填写：案件名、客户名、一句话案情（可选：手动选案件类型）
4. 系统创建案件 → 进入案件详情页（空状态）
5. 看到引导："导入第一个素材"——粘贴聊天/上传截图/上传 PDF/语音口述
6. 导入素材 → AI 开始分析（loading 状态）
7. AI 返回结果 → 逐条展示，律师确认/修改/删除
8. 确认后时间线和待办更新
9. 返回首页 → 看到今日概览

## 老用户日常使用流程

1. 打开应用 → 首页显示今日概览
2. 查看待办和提醒
3. 如有新素材 → 点击"导入素材" → 选择案件 → 上传
4. 查看 AI 分析结果 → 确认入库
5. 点击具体案件查看进展
6. 处理待办事项

## 创建案件流程

1. 首页点击"+ 新建案件"
2. 弹出创建表单：案件名（必填）、客户名（必填）、一句话案情（必填）、案件类型（可选，AI自动推荐）
3. 提交 → 系统创建案件
4. 跳转案件详情页
5. 自动弹出"导入素材"引导

## 导入素材流程

1. 在案件详情页点击"导入素材"
2. 选择导入方式：
   - 粘贴文本 → 文本框 + 粘贴
   - 上传截图 → 打开文件选择器
   - 上传 PDF → 打开文件选择器
   - 语音口述 → 开始录音
3. 提交 → 素材进入分析队列
4. 显示 AI 分析进度
5. 分析完成 → 跳转"AI 结果待确认"页面
6. 逐条确认 AI 提取结果
7. 确认完毕 → 时间线和待办更新

## 查看案件详情流程

1. 首页点击案件卡片
2. 进入案件详情页，默认显示"概览" Tab
3. 可切换到：时间线 / 待办 / 证据 / 沟通记录 / AI 建议
4. 每个 Tab 内容可快速扫读

## 确认 AI 结果流程

1. 进入"待确认"列表（可从首页或案件页进入）
2. 看到 AI 提取的每条结果：
   - 类型标签（时间事件 / 证据 / 待办 / 日期 / 风险）
   - AI 提取内容
   - 原始素材引用
   - 置信度标记
3. 三个操作按钮：✓ 确认 / ✎ 修改 / ✕ 删除
4. 点击"修改"→ 变为可编辑状态 → 保存
5. 确认完毕的结果进入案件档案

## 编辑案件流程

1. 案件详情页点击"编辑"
2. 修改案件基本信息
3. 保存

## 删除 / 归档流程

1. 案件列表页 → 右键/长按 → "删除"或"归档"
2. 弹出二次确认对话框
3. 确认后执行操作

## 异常流程

### 页面加载失败

用户看到：错误提示 + 重试按钮
用户可以：点击重试 / 返回首页

### 数据为空

用户看到：空状态插画 + 说明文字 + 引导操作
用户可以：按引导开始操作

### 操作失败

用户看到：具体错误原因 + 建议操作
用户可以：重试 / 取消

### AI 生成失败

用户看到："AI 分析失败"提示 + 可能原因
用户可以：重新分析 / 手动录入

### AI 结果不确定

用户看到：低置信度标记 + 原始素材引用
用户可以：查看原文判断 / 修改 / 删除
