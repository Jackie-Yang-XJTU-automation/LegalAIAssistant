# UI Style Guide — 律师私人案件助理

## 产品气质

关键词：
- **专业可信**：律师对工具的信任感
- **清晰高效**：信息层级明确，快速扫读
- **沉稳克制**：不过度装饰，聚焦内容本身

## 视觉原则

- 清晰优先于美观
- 信息层级明确，重要信息突出
- 移动端优先，桌面端增强
- 避免过度装饰，减少视觉噪音
- 重要状态（风险、到期）始终可见
- 操作反馈明确，AI 结果可辨识

## 色彩

### 主色

深蓝色系，传达专业、可信、沉稳。

- Primary: `#1E3A5F`（深蓝，导航、主要按钮）
- Primary Light: `#2D5A8E`（浅蓝，hover 状态）
- Primary Pale: `#EBF0F7`（极浅蓝，背景）

### 辅助色

- Secondary: `#6B7D8E`（灰蓝，次要信息）
- Secondary Light: `#F5F7FA`（浅灰蓝，卡片背景）

### 状态色

- Success: `#2E7D32`（绿色，已确认、已完成）
- Warning: `#ED6C02`（橙色，即将到期、需注意）
- Error: `#D32F2F`（红色，已逾期、高风险）
- Info: `#0288D1`（蓝色，信息提示）

### AI 标记色

- AI Generated: `#7B5EA7`（紫色，标识 AI 生成内容）
- AI Low Confidence: `#FFF3E0`（浅橙背景，低置信度）

## 字体

### 标题

- 使用系统默认中文字体（PingFang SC / Microsoft YaHei）
- 标题层级：H1 24px, H2 20px, H3 16px
- 加粗，深色

### 正文

- 14px，行高 1.6
- 颜色 `#333333`
- 最大行宽 72ch

### 数字 / 日期

- 使用等宽数字字体变体（Tabular Numbers）
- 16px，加粗
- 颜色 `#1E3A5F`

## 布局

- 移动端优先设计
- 页面最大宽度：桌面端 1200px，内容区 800px
- 重要操作靠近用户当前上下文
- 列表信息可快速扫读（标题 + 摘要 + 状态 + 时间）
- 详情页突出状态、时间、下一步动作
- 卡片间距 16px
- 页面内边距 16px（移动端）/ 24px（桌面端）

## 交互原则

- 所有操作必须有即时反馈
- 危险操作（删除案件）必须二次确认
- AI 生成内容统一使用紫色标识，与手动录入区分
- AI 生成结果必须允许用户确认/修改/删除
- 错误提示必须告诉用户下一步能做什么
- 加载状态使用骨架屏，不给空白
- 表单提交防重复点击
