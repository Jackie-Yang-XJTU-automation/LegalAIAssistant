## Components

### Button

#### Variants

- **primary**: 深蓝背景白字，用于主要操作（创建、保存、确认）
- **secondary**: 白底蓝边蓝字，用于次要操作（取消、返回）
- **ghost**: 无边框无背景，用于低优先级操作
- **danger**: 红色，用于删除等危险操作

#### Sizes

- sm: 28px 高，12px 字号
- md: 36px 高，14px 字号
- lg: 44px 高，16px 字号

#### States

- default / hover / active / disabled / loading

---

### Card

用于承载核心信息块（案件卡片、待办卡片、素材卡片）。

要求：
- 白色背景，低阴影
- 标题清晰（16px 加粗）
- 状态标签可见（彩色圆点 + 文字）
- 主要操作在右下角
- 支持点击跳转
- 移动端全宽

---

### List Item（案件列表项）

```
┌──────────────────────────────────┐
│ ● 进行中    张三民间借贷案        │
│ 最新：2025-03-20 拟起诉状         │
│ 📋 3 待办  ⚠ 1 即将到期          │
│ 更新于 2 小时前            →     │
└──────────────────────────────────┘
```

必须包含：
- 状态指示器
- 案件名称（加粗）
- 最新进展摘要（一行）
- 待办数量 + 到期提醒数量
- 最后更新时间
- 可点击区域

### Status Badge

- 进行中：绿色圆点 + "进行中"
- 待确认：橙色圆点 + "待确认"
- 已逾期：红色圆点 + "已逾期"
- 已结案：灰色圆点 + "已结案"
- 已归档：灰色 + "已归档"

### Empty State

```
┌──────────────────────────────────┐
│          📋                       │
│     还没有案件                    │
│   创建您的第一个案件              │
│   开始用 AI 管理案件进展          │
│                                   │
│      [+ 创建案件]                 │
└──────────────────────────────────┘
```

必须包含：
- 空状态图标
- 简短说明
- 引导操作按钮

---

### Error State

```
┌──────────────────────────────────┐
│          ⚠                       │
│     加载失败                      │
│   网络连接异常，请检查网络        │
│                                   │
│      [重试]                       │
└──────────────────────────────────┘
```

必须包含：
- 错误原因
- 用户可执行动作（重试/返回）
- 重试入口

---

### Loading State

用于：
- 页面加载：骨架屏（灰色占位块）
- 局部模块加载：小 spinning 图标
- AI 生成中：动画 + "AI 正在分析..." + 预计步骤
- 表单提交中：按钮 loading 状态
